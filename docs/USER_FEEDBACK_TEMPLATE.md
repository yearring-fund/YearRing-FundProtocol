# USER_FEEDBACK_TEMPLATE.md — Step3 用户反馈收集模板

生成时间：2026-04-05
适用阶段：Step3（白名单受邀试运营）

---

## 使用说明

本模板用于收集白名单用户在 Step3 运行期间的使用反馈，供团队了解体验质量、发现问题并决策是否扩容。
反馈收集方式：由 ADMIN 在每次操作协助后或定期（每 2 周）主动询问用户并填写。
文件命名：`evidence/user_feedback_<YYYYMMDD>_<user_id>.md`

---

## 基本信息

| 字段 | 填写 |
|---|---|
| 反馈编号 | FB-001（递增） |
| 收集时间 | YYYY-MM-DD |
| 用户标识 | User A / User B（不记录真实钱包地址） |
| 参与时长 | 自 YYYY-MM-DD 起 |
| 本次反馈触发 | 定期收集 / 存款后 / 赎回后 / 用户主动 |
